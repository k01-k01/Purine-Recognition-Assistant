# 痛风患者食物嘌呤识别H5应用产品需求文档

## 1. 产品概述

### 1.1 产品背景
根据《成人高尿酸血症与痛风食养指南（2024年版）》，我国成人居民高尿酸血症患病率为14%，痛风患病率为0.86%~2.20%。高尿酸血症与痛风的发生与膳食密切相关，合理控制膳食中的嘌呤摄入对疾病管理至关重要。

### 1.2 产品定位
本产品是一款基于AI图像识别技术的H5应用，旨在帮助痛风患者快速识别食物嘌呤含量，管理日常饮食，记录尿酸数据，从而更好地控制病情。

### 1.3 目标用户
- 痛风患者  
- 高尿酸血症患者  
- 关注饮食健康的用户  

### 1.4 核心价值
- 快速准确识别食物嘌呤含量  
- 提供科学的饮食建议  
- 便捷的尿酸数据管理  
- 基于权威指南的食物数据库  

---

## 2. 功能需求

### 2.1 整体架构
应用采用H5技术开发，底部导航栏包含3个主要Tab：  
1. **识别**：AI拍照识别食物嘌呤  
2. **食物**：查看分类食物嘌呤数据  
3. **我的**：尿酸记录与个人数据管理  

### 2.2 识别页面
- **功能流程**：  
  1. 用户点击“拍照”或“从相册选择”上传食物图片。  
  2. 调用通义千问VL模型识别食物类型。  
  3. 返回识别结果：  
     - 食物名称  
     - 嘌呤含量等级（高/中/低）  
     - 饮食建议（如“需限制摄入”）  
  4. 支持重新拍摄或编辑识别结果。  
- **技术限制**：  
  - 图片大小不超过2MB，支持JPG/PNG/WEBP格式。  

### 2.3 食物页面
- **功能模块**：  
  1. **筛选栏**：支持按嘌呤含量（高/中/低）快速筛选。  
  2. **食物列表**：  
     - 每项展示食物图片、名称、嘌呤含量值及描述（如“动物内脏，高嘌呤”）。  
     - 列表支持滚动加载更多。  
- **数据来源**：本地静态JSON文件，初版包含100+常见食物数据。  

### 2.4 我的页面
- **尿酸折线图**：  
  - 展示用户历史尿酸值变化趋势（时间范围为最近30天）。  
  - 支持点击数据点查看详情。  
- **数据统计区**：  
  - 显示近3次记录的尿酸平均值、最低值、最高值。  
- **记录列表**：  
  - 按时间倒序展示所有尿酸记录（日期、尿酸值、备注）。  
  - 支持新增或删除记录。  
- **数据存储**：所有记录保存至localStorage。  

---

## 3. 技术方案

### 3.1 前端技术
- **框架**: Next.js 14 (React 18)  
- **UI组件库**: Tailwind CSS + shadcn/ui  
- **图表库**: ECharts（折线图渲染）  
- **状态管理**: Zustand  
- **网络请求**: Axios  

### 3.2 AI识别服务
- **模型**: 通义千问VL模型（RESTful API调用）  
- **图片处理**: 前端压缩至2MB以下，格式校验。  

### 3.3 数据存储
- **尿酸记录**: localStorage  
- **食物数据**: 静态JSON（初版），未来可迁移至IndexedDB。
